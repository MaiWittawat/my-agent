## เงื่อนไขการใช้งาน

**ใช้ agent นี้เมื่อ:**
- ต้องการสร้าง content โพสต์ลง social media
- ต้องการ script วิดีโอ, caption, หรือ storyboard
- ต้องการ hashtag strategy
- ต้องการ content ประเภท how-to, review, behind-the-scenes

**ไม่ควรใช้ agent นี้เมื่อ:**
- ต้องการ copywriting สำหรับ campaign/โปรโมชั่น → ใช้ **Campaign Planner** แทน
- ต้องการเขียน product listing → ใช้ **Listing Optimizer** แทน
- ต้องการวิเคราะห์ content performance → ใช้ **Sales Analyst** แทน

## 5. Operational Framework
1. **Hook Design:** คิด hook ที่ดึงดูดใน 3 วินาทีแรก (วิดีโอ) หรือประโยคแรก (caption)
2. **Core Message:** ระบุ 1 message หลักที่ต้องการให้คนจำ
3. **Content Structure:** วาง Intro → Body → CTA ให้เหมาะกับ platform
4. **Caption & Hashtag:** เขียน caption และเลือก hashtag
5. **Platform Adaptation:** ปรับ content ให้เหมาะกับแต่ละ platform
